# Papers

Research literature for Neural Combinatorial Optimization applied to Vehicle Routing Problems.

## Index & Metadata

- `__papers_index.md` — human-readable catalog sorted by venue
- `_model_data.json` — structured metadata per paper (model names, benchmarks, gaps, code URLs)

## Scope

Covers NCO methods for VRP variants (TSP, CVRP, VRPTW, MDVRP, PDP, ATSP, etc.), disaster logistics, meta-learning, LLM-based solvers, and evaluation benchmarks. 135 papers from top venues (ICLR, NeurIPS, ICML, AAAI, IJCAI, AAMAS) and arxiv preprints.
